Olsfors Elektriska — Widget Demo

Detta är en demosida för embed-widgeten. Klicka på chat-bubblan nere till höger för att öppna.

Widgeten laddas via React + Vite. Kör npm run build:embed för att producera en single-fil drop-in version (dist-embed/widget.js) som customers embeddar med en <script>-tagg.
